## 本WIKI现已支持

### 禁用指定页面的编辑链接

```yaml
---
editLink: false
---
```

### 导航栏/侧边栏相关

```yaml
---
navbar: false # 禁用导航栏
sidebar: false # 禁用侧边栏
sidebarDepth: 2 # 同时提取 h2 和 h3 标题(值为0-2,0为禁用)
---
```
